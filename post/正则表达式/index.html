|字数总计:4.9k|阅读时长:21分钟

基本介绍

一个正则表达式，就是用某种模式去匹配字符串的一个公式。

语法

转义号\\

\\符号 说明:在我们使用正则表达式去检索某些特殊字符的时候，需要用到转义符号，否则检索不到结果，甚至会报错的。

需要用到转义符号的字符有以下: . * + ( ) $ / \ ? [ ] ^ { }

字符匹配符

选择匹配符

在匹配某个字符串的时候是选择性的，即:既可以匹配这个，又可以匹配那个，这时需
要用到选择匹配符号|

限定符

用于指定其前面的字符和组合项连续出现多少次

定位符

定位符, 规定要匹配的字符串出现的位置，比如在字符串的开始还是在结束的位置.
