动态类型的问题

js 属于动态类型语言，例如

obj 可能只是个字符串

obj 也有可能是个函数

obj 类型不确定，就给后期使用者带来了麻烦，一旦参数传不对，代码就崩溃了

动态类型意味着

  • 运行代码时才知道发生什么 (running the code to see what happens)

静态类型意味着

  • 在代码运行前，就对它的行为做出预测 (make predications about what code is expected before it runs)
